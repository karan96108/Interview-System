# Crisp — AI-Powered Interview Assistant

The `app/` directory hosts a React + Redux implementation of an AI-assisted interview experience. Candidates upload their resume, complete a timed six-question interview, and receive automated scoring. Interviewers get a synchronized dashboard with transcripts, scores, and summaries.

## Repository Layout

```
.
├── app/        # React + TypeScript source (see app/README.md for details)
└── docs/       # Assignment brief
```

Navigate into `app/` for setup, features, and deployment instructions.
